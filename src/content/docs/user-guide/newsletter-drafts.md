# Newsletter Drafts

This is where curation pays off.

## Assembling a Draft
Drafts are built by composing:
1. High-Relevance fresh articles.
2. Promoted Theme Suggestions.
3. Editor-generated Original Content Ideas.

## Reordering
The Draft builder allows you to drag-and-drop these artifacts, rewriting transition languages automatically where necessary.

## Exporting
Click Export to generate raw Markdown or HTML ready to be pasted into Mailchimp, Substack, Hackernews, or a ghost CMS.

## Iterating with Feedback
If the synthesized draft output misses your tone, utilize the thumbs up/down mechanisms on individual sections to train the system.
